# PliegoRS identity assets

This directory is the source of truth for the PliegoRS symbol. Product favicons
and starter icons are exports of the same geometry.

## Files

- `pliegors-symbol.svg`: transparent symbol for light surfaces.
- `pliegors-symbol-reversed.svg`: transparent symbol for dark surfaces.
- `pliegors-app-icon.svg`: self-contained application icon.

The symbol is a folded path with an independent red terminal square. Do not
round, rotate, stretch, recolor individual segments, add effects, or rebuild it
from a typeface. Keep clear space equal to the red square on every side. Use the
application icon at 32 CSS pixels or larger and the standalone symbol at 24 CSS
pixels or larger.

## Core colors

| Name | Value | Role |
| --- | --- | --- |
| Carbon | `#171916` | Dark field and light-surface symbol |
| Paper | `#f3f4ee` | Reversed symbol |
| Terminal | `#c23a30` | Terminal square only |
